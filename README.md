# SwagLabs Selenium Automation project

## Description:
Swag Labs is a testing website. This project is the result of a three-month intensive QA course. 
After completing manual testing and bug reporting, I selected several test cases for automation.


## Tech Stack:
 - Java (OpenJDK 26)
 - IntelliJ IDEA 2026.1
 - Selenium WebDriver 4.43.0
 - TestNG
 - Maven
 - Firefox browser
 - Run on Windows 11 Pro

## Prerequisites:
 - OpenJDK 26
 - Apache Maven

## Installation

Clone the repository in IDE terminal:


```bash
git clone https://github.com/AleksandraPrlincevic/projekatITBootcamp.git
```
## How to run:
Open the project in IntelliJ IDEA and run test suites directly from the IDE.
